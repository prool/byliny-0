Файл circle30bpl18.zip взят с сайта МПМ "Былины" mud.ru
Изначально этот файл предсталял собой CircleMUD 3.0b pl 18,
русифицированный авторами Былин и с начальной зоной Былин.
По сути это первая (нулевая) версия Былин, написанная еще на чистом
C без вставок C++ и без использования пакета BOOST.

Я собрал его для Винды в среде cygwin (но для запуска пакет cygwin
не нужен!) и внес несущественные сервисные правки.

На консоли русские буквы не отображаются, но в самом маде все нормально

Запускать в Виндовс так

mud.bat

Брандмауэр Виндовс спросит, разрешать ли программе доступ к Интернету. Разрешайте,
это безопасно. Кроме того, без разрешения сможете играть только вы, а с разрешением
к вашему маду смогут коннектиться и другие игроки

Входить мад-клиентом JMC так:

#con localhost 4000

Я собрал его под Debian 6.0, внеся при необходимости коррективы в Makefile,
а также в исходники (в которых свои правки я помечал комментарием "prool")

В Ubuntu и Centos тоже собирается без проблем

Зачем: а просто так. Может, кому пригодится
